사용 팁

효과적인 활용 방법

1. 회의 음성 → 텍스트 변환 (STT)

회의 진행 중 발생하는 음성을 실시간으로 텍스트로 변환하여 동시에 화면에 표시합니다. 회의 종료 후 따로 파일을 업로드하지 않아도, 회의와 동시에 자동 회의록이 생성됩니다.

주요 특징
  • 실시간 음성 스트리밍 및 텍스트 변환
  • 발화자별 구분으로 "누가 무슨 말을 했는지" 명확히 표시
  • 변환된 텍스트는 자동으로 데이터베이스에 저장

예시:
[홍길동] 이번 분기 예산을 논의하겠습니다.
[김영희] 네, 2억 원 정도 필요합니다.

2. 회의록 저장 및 조회

생성된 회의록을 체계적으로 관리하고, 리스트 및 상세 페이지에서 확인할 수 있습니다. 필요한 회의록은 날짜/키워드로 검색할 수 있으며, PDF 형식으로 내보낼 수 있습니다.

주요 특징
  • 회의 종료 시 회의록 자동 저장
  • 리스트: 제목, 일자, 참석자, 요약 표시
  • 상세: 전체 대화, 요약, 액션 아이템 확인
  • Markdown → PDF 변환 지원

예시:
회의록 제목: 9월 24일 마케팅 전략 회의
참석자: 홍길동, 김영희, 박철수
요약: 캠페인 예산 2억 원 확정, 기획안 작성 필요

3. 회의록 기반 검색 챗봇

누적된 회의록을 대상으로 사용자가 질문하면, 챗봇이 관련 내용을 찾아줍니다. 단순 키워드 검색을 넘어, 자연어 질의응답을 지원합니다.

주요 특징
  • 회의록 데이터를 DB/검색 인덱스로 저장
  • 자연어 질문 입력 시 관련 회의록 검색
  • 챗봇이 답변 요약 생성

예시:
Q: 지난달 예산 관련 논의 뭐였어?
A: 8월 12일 회의에서 마케팅 예산을 1.5억 원으로 조정하기로 합의했습니다.

4. 사용자 맞춤 기능

사용자의 역할·관심사에 따라 회의록 요약을 맞춤 제공하고, 자주 쓰는 용어나 약어를 자동으로 풀어서 보여줍니다.

주요 특징
  • 사용자 프로필(부서, 직무) 등록
  • 관련된 내용만 추려 별도 요약 제공
  • 주요 약어·전문 용어 자동 해석

예시 - 맞춤 요약:
[마케팅 담당자용 요약]
- 프로모션 일정: 10월 시작
- 예산: 2억 원 확정

예시 - 용어 해석:
Q: LTV가 뭐예요?
A: LTV는 고객 생애 가치(Life Time Value)의 약어로, 고객이 우리 회사와 관계를 유지하는 동안 발생시키는 총 수익을 의미합니다.

5. 하이라이트 (Highlight) 기능

사용자가 회의 중(또는 이전)에 설정한 특정 단어 및 문장을 추적 및 탐색해 하이라이트 처리합니다. 회의 종료 후에는 하이라이트만 모아보기를 제공합니다.

주요 특징
  • 사용자가 선택한 문장 또는 키워드 지정
  • 지정된 텍스트를 실시간 탐지
  • 해당 구간을 태그 처리하여 회의록에 표시
  • 종료 후 "하이라이트 모음집" 생성

예시:
[Highlight] 다음 분기 예산은 2억 원으로 확정

6. 캘린더 연동

회의록에서 일정이나 업무 관련 문장을 인식하여, 자동으로 캘린더 이벤트나 업무 할당으로 등록합니다.

주요 특징
  • 일정 자동 인식 및 캘린더 이벤트 생성
  • 생성 전 사용자에게 확인 절차 제공
  • 사용자 설정에 따라 수정 가능

예시:
캘린더: 10월 2일 오전 10시 – 마케팅 전략 회의
업무: [김영희] 보고서 초안 작성

효과적인 서비스 활용 방법

📝 회의 전 준비
  • 개인정보 설정에서 직무를 미리 선택해두면 맞춤형 요약을 받을 수 있습니다
  • 중요하게 추적하고 싶은 키워드를 미리 설정해보세요
🎙️ 회의 중 활용
  • 명확한 발화를 위해 한 번에 한 사람씩 말하도록 권장합니다
  • 중요한 결정사항이나 액션 아이템은 강조하여 말하면 더 잘 인식됩니다
  • 실시간으로 표시되는 텍스트를 확인하며 필요시 수정 요청하세요
📊 회의 후 관리
  • 회의록 상세 페이지에서 하이라이트된 내용을 먼저 확인하세요
  • PDF로 내보내기하여 팀원들과 공유할 수 있습니다
  • 검색 챗봇을 활용하여 이전 회의 내용을 빠르게 찾아보세요
  • 캘린더 연동 기능으로 일정을 자동 등록하세요

  • "김영희님이 맡은 업무가 뭐였지?"와 같이 담당자 기준으로도 검색 가능합니다
  • 전문 용어가 이해되지 않을 때는 챗봇에게 바로 물어보세요
